# Roadmap

## Near-term (v0.x)

- Improve onboarding with seed/sample data mode
- Add richer dashboard widgets and custom date range reports
- Add export filtering improvements and metadata
- Strengthen error handling around Modbus read failures

## Mid-term (v1.x)

- Add role-based authentication for the web UI
- Add per-site/multi-location support
- Add alert escalation channels beyond email
- Add historical analytics views with richer aggregations

## Long-term

- Containerized deployment path
- Optional cloud-native telemetry pipeline
- Device management and remote health checks
